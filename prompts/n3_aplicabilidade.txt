Extrator RASE N3 (aplicabilidade).
Use APENAS o operador N2 de aplicabilidade para gerar properties_n3.
Texto completo e Texto N1 sao apenas referencia de desambiguacao.

Formato obrigatorio de saida:
- Retorne SOMENTE um JSON valido com as chaves:
  type, object, property, comparation, target, unit
- Nao retorne markdown nem texto extra.

Regras:
1) Se o operador N2 estiver vazio, retorne todos os campos vazios.
2) Se o operador N2 NAO estiver vazio:
   - type deve ser "aplicabilidade".
   - se nao houver informacao suficiente para os demais campos, mantenha somente type e deixe os demais vazios.
3) object: substantivo principal do escopo (ou varios separados por "; ").
4) Quando houver classificacao clara de uso/tipo, preencha property/comparation/target.
5) unit so deve ser preenchido quando houver unidade explicita.

Exemplo 1:
Operador N2 (aplicabilidade):
"As edificacoes residenciais multifamiliares, condominios e conjuntos habitacionais"
Resposta:
{{
  "type": "aplicabilidade",
  "object": "edificacao",
  "property": "uso",
  "comparation": "=",
  "target": "residenciais multifamiliares; condominios; conjuntos habitacionais",
  "unit": ""
}}

Exemplo 2:
Operador N2 (aplicabilidade):
"Nas edificacoes e equipamentos urbanos"
Resposta:
{{
  "type": "aplicabilidade",
  "object": "edificacao; equipamento urbano",
  "property": "",
  "comparation": "",
  "target": "",
  "unit": ""
}}
