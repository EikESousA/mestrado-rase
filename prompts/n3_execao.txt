Extrator RASE N3 (execao).
Use APENAS o operador N2 de execao para gerar properties_n3.
Texto completo e Texto N1 sao apenas referencia de desambiguacao.

Formato obrigatorio de saida:
- Retorne SOMENTE um JSON valido com as chaves:
  type, object, property, comparation, target, unit
- Nao retorne markdown nem texto extra.

Regras:
1) Se o operador N2 estiver vazio, retorne todos os campos vazios.
2) Se o operador N2 NAO estiver vazio:
   - type deve ser "execcao".
   - quando for apenas condicao curta (ex.: "em reformas", "se ..."), mantenha type preenchido
     e deixe os demais campos vazios se nao houver atributo claro.
3) Quando houver informacao clara de uso/tipo excluido, preencha property/comparation/target.
4) unit so deve ser preenchido quando houver unidade explicita.

Exemplo 1:
Operador N2 (execao):
"Areas de uso restrito, como casas de maquinas e barriletes"
Resposta:
{{
  "type": "execcao",
  "object": "espaco",
  "property": "uso",
  "comparation": "contém",
  "target": "restrito",
  "unit": ""
}}

Operador N2 (execao):

  "type": "execcao",

  "property": "",

  "unit": ""
