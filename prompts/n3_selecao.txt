Extrator RASE N3 (selecao).
Use APENAS o operador N2 de selecao para gerar properties_n3.
Texto completo e Texto N1 sao apenas referencia de desambiguacao.

Formato obrigatorio de saida:
- Retorne SOMENTE um JSON valido com as chaves:
  type, object, property, comparation, target, unit
- Nao retorne markdown nem texto extra.

Regras:
1) Se o operador N2 estiver vazio, retorne todos os campos vazios.
2) Se o operador N2 NAO estiver vazio:
   - type deve ser "selecao".
   - mesmo que seja apenas qualificador curto (ex.: "internos", "externos", "eventuais"),
     mantenha type preenchido e deixe os demais campos vazios quando nao houver semantica clara.
3) Quando houver classificacao de uso/tipo, preencha property/comparation/target.
4) unit so deve ser preenchido quando houver unidade explicita.

Exemplo 1:
Operador N2 (selecao):
"uso publico ou coletivo"
Resposta:
{{
  "type": "selecao",
  "object": "espaco",
  "property": "uso",
  "comparation": "=",
  "target": "publico ou coletivo",
  "unit": ""
}}

Exemplo 2:
Operador N2 (selecao):
"internos"
Resposta:
{{
  "type": "selecao",
  "object": "",
  "property": "",
  "comparation": "",
  "target": "",
  "unit": ""
}}
